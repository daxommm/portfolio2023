병원의 깔끔하고 부드러운 느낌을 위하여 이미지와 텍스트로만 표현된 정적인 섹션에 animation을 넣어 자연스럽게 등장하도록 하였고, 마우스 호버 효과에서 transition을 넣어 끊기는 느낌이 없이 부드럽게 노출되도록 적용하였습니다.
인트로 페이지에 유튜브 영상을 풀페이지로 삽입하여 영상 비율이 깨지지 않도록 반응형 작업을 하였고, 웹 / 모바일 환경에서 동일하게 무한 재생, 자동 재생되도록 구현하였습니다.
재생 상황을 노출시켜주는 progress bar를 이용하여 어느 지점이 재생되고 있는지 안내시켜주며, 재생버튼과 일시정지 버튼으로 슬라이드를 제어할 수 있게 하였습니다. 또한, 사용자들이 내용을 충분히 볼 수 있을 정도의 속도로 autoplay를 적용하였으며, slideLeft 효과로 자연스럽게 넘어가는 슬라이드를 구현하였습니다.
고정 메뉴를 사용하여 스크롤 시 특정 영역에 맞춰 위치가 고정이 되며, 스크롤이 해당 영역에 위치해 있음을 확인할 수 있도록 링크에 클래스를 붙여서 효과를 주었습니다. 또한 링크를 클릭하면 해당 영역으로 부드럽게 이동이 됩니다.
